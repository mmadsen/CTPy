def ctpy.math.trait_statistics.TraitStatisticsPerSample.process_trait_statistics (   self)

Given a sample in the fulldataset, we want to calculate the following: trait richness per locus for sample, mean trait richness per generation for sample, shannon entropy and iqv per locus for sample, mean shannon entropy and iqv per generation for sample.

These statistics are saved to the database with sim run ID, replication, sample size, mutation rate, simulation time, and population size as keys. With these keys, we can match them up to their post-classification counterparts.

:return: no return value
